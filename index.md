# 快譯 - 右鍵即時翻譯 隱私權政策

**最後更新日期：** 2026年6月6日

本擴充功能（「快譯」）非常尊重用戶隱私。

## 1. 資料收集與使用
- 本擴充功能**不會收集、儲存或分享**任何個人資料。
- 唯一傳送至外部嘅資料，只有用戶**主動選取嘅文字**，用於即時翻譯。
- 翻譯完成後，資料不會被保留。

## 2. 第三方服務
- 使用 Google 公開翻譯 API（https://translate.googleapis.com）進行翻譯。
- 所有請求均由用戶瀏覽器直接發出，本擴充功能不會中間儲存資料。

## 3. 權限說明
- `contextMenus`：僅用於顯示右鍵選單。
- `<all_urls>`：僅用於在當前網頁顯示翻譯彈窗。
- 所有功能均在本機運行。
